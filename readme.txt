唐诗宋词(chinese poem)

== Description ==

<p>This plug-in can make you show one piece of Chinese Tangsong Poetics random in your Wordpress system every time.First ,you can install it and then add it in the widgets page . </p>

<p>这个插件允许在Wordpress中随机显示唐诗宋词。如果使用widget，也可以在widget的配置窗口自定义模块。</p>

<h3>可用模块</h3>
<ul>
<li>caigentan:   菜根谭</li>
<li>dao:	     道德经</li>
<li>joke:	     笑话集</li>
<li>lunyu:	     论语</li>
<li>maoshici:    毛泽东诗词</li>
<li>maoyulu:     毛泽东语录</li>
<li>proverb:     谚语</li>
<li>song100:     宋诗100首</li>
<li>songproses:  宋词</li>
<li>tang300:     唐诗300首</li>
<li>zengguang:   增广贤文</li>
</ul>
<p>诗词原始资料来源于<a href="http://code.google.com/p/chinese-fortune/" title="中文fortune" >chinese-fortune</a></p>

== Installation ==

1. Upload the Plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Add Anywhere:  <?php show_chinese_poem() ?>
4. If you need to configure the CSS style of 唐诗宋词(chinese poem), open and edit: 
	/wp-content/plugins/chinese-poem/poem.css
